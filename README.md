# Jenkins Inbound Agents

Lightweight and extensible Jenkins inbound agent Docker images built with a focus on CI/CD pipelines, security, and reproducibility.

Supports multiple variants built from dedicated Dockerfiles.

## :whale: Available Images

- [`t0mmili/jenkins-inbound-agent-docker`](https://hub.docker.com/r/t0mmili/jenkins-inbound-agent-docker) – inbound agent with Docker CLI  
![Docker Pulls](https://img.shields.io/docker/pulls/t0mmili/jenkins-inbound-agent-docker)
![Docker Image Size](https://img.shields.io/docker/image-size/t0mmili/jenkins-inbound-agent-docker)

- [`t0mmili/jenkins-inbound-agent-trivy`](https://hub.docker.com/r/t0mmili/jenkins-inbound-agent-trivy) – inbound agent with Trivy scanner  
![Docker Pulls](https://img.shields.io/docker/pulls/t0mmili/jenkins-inbound-agent-trivy)
![Docker Image Size](https://img.shields.io/docker/image-size/t0mmili/jenkins-inbound-agent-trivy)

Each image is versioned and also published with the `latest` tag.
